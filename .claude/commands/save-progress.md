# Save Progress

현재 세션의 작업 진행상황을 저장해주세요.

## 수행할 작업

1. 현재 프로젝트 루트를 확인합니다 (git root 또는 현재 디렉토리)
2. 이번 세션에서 수행한 작업을 분석합니다:
   - 어떤 작업을 했는지
   - 어떤 파일을 수정/생성했는지
   - 어떤 것이 완료되었고 어떤 것이 진행 중인지
   - 다음에 해야 할 것은 무엇인지
3. `.claude/WORK_IN_PROGRESS.md` 파일을 생성/업데이트합니다

## 저장 형식

아래 형식으로 저장합니다:

```markdown
---
saved_at: [현재 ISO 날짜시간]
session_id: [현재 세션 ID가 있으면]
project_dir: [프로젝트 경로]
status: in_progress
---

# Work In Progress

## 현재 작업
[이번 세션에서 하고 있던 핵심 작업 설명]

## 완료된 항목
- [x] [완료된 것들을 체크리스트로]

## 진행 중인 항목
- [ ] [아직 완료되지 않은 것들과 진행률]

## 다음 할 일
1. [우선순위 순서로 다음에 해야 할 것]

## 핵심 컨텍스트
[이어서 작업할 때 반드시 알아야 할 정보]
- 주요 결정사항
- 주의사항/블로커
- 참고할 에러나 이슈

## 관련 파일
- [수정한 주요 파일 경로들]
```

## 규칙
- `git diff --name-only`로 변경된 파일 목록을 확인합니다
- 작업 내용은 구체적이고 간결하게 작성합니다
- 저장 완료 후 사용자에게 안내합니다
- 기존 파일이 있으면 완전히 덮어씁니다
